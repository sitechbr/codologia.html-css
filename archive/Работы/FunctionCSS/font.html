Font

Описание Универсальное свойство, которое позволяет одновременно задать несколько характеристик шрифта и текста. Значения В качестве обязательных значений свойства font указывается размер шрифта и его семейство. Остальные значения являются опциональными и задаются при желании. Для подробного ознакомления смотрите информацию о каждом свойстве отдельно. Пример: <style> .layer1 { font: 12pt sans-serif; } h1 { font: 200% serif; } </style>
